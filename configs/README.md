# Required Keys for Configuration JSON file

An example Configuration file is provided at `classifier_configs.py`

## Required top level Keys

    'NAME'
    "SEED"
    'USE_CUDA'
    'CUDNN_DETERMINISTIC'
    'CUDNN_BENCHMARK'
    'N_GPU'
    'GPU_DEVICE'
    'ARCH'
    'DATASET'
    'DATALOADER'
    'OPTIMIZER'
    'LOSS'
    'METRICS'
    'LR_SCHEDULER'
    'TRAINER'
    'LOGGER'
